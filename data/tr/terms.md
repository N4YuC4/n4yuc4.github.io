# Kullanım Koşulları

N4YuC4 Kişisel Web Sitesi'ni ziyaret ederek ve kullanarak, aşağıda belirtilen kullanım koşullarını kabul etmiş olursunuz. Bu web sitesi kişisel bir portfolyo ve bilgilendirme sitesidir.

## 1. Koşulların Kabulü

Bu web sitesini kullanarak bu koşulları okuduğunuzu ve anladığınızı kabul etmiş olursunuz. Eğer kabul etmiyorsanız, lütfen siteyi kullanmaktan kaçının.

## 2. İçerik Mülkiyeti ve Kullanımı

Bu web sitemizdeki metinler, projeler ve görseller de dahil olmak üzere tüm içerik (aksi belirtilmedikçe) web sitesi sahibine aittir ve telif hakkı yasalarıyla korunmaktadır.

*   **Kişisel Kullanım:** İçeriği kişisel, ticari olmayan bilgilendirme amaçlı olarak görüntüleyebilir ve okuyabilirsiniz.
*   **Yasaklanmış Kullanım:** Web sitesi sahibinin açık yazılı izni olmaksızın içeriği kopyalayamaz, yeniden dağıtamaz veya ticari amaçlarla kullanamazsınız.
*   **Atıfta Bulunma:** İçeriği yasal sınırlar dahilinde paylaştığınızda veya referans gösterdiğinizde, web sitesi sahibine net bir şekilde atıfta bulunmanız gerekir.

## 3. Sorumluluk Reddi

Bu web sitesindeki bilgiler yalnızca genel bilgilendirme amacıyla \"olduğu gibi\" sunulmaktadır.

*   **Doğruluk:** Doğruluk için çaba göstersem de, web sitesi sahibi bilgilerin eksiksizliği veya güncelliği konusunda hiçbir garanti vermez.
*   **Teknik Sorunlar:** Web sitesi sahibi, bu web sitesinin kullanılmasından veya harici bağlantıların takip edilmesinden kaynaklanabilecek herhangi bir teknik sorundan, virüsten veya hasardan sorumlu değildir.

## 4. Harici Bağlantılar

Bu site Medium, LinkedIn ve GitHub gibi harici platformlara bağlantı vermektedir. Web sitesi sahibi, bu üçüncü taraf sitelerin içeriğinden veya kullanılabilirliğinden sorumlu değildir.

## 5. Koşullarda Değişiklik Yapılması

Web sitesi sahibi, bu koşulları herhangi bir zamanda güncelleme hakkını saklı tutar.
