Who Am I?

Hi! I'm Nazmi Yücel Çan. I am an Information Systems and Technologies graduate and a Machine Learning Engineer / Data Scientist focused on deploying and optimizing AI models in production environments. I co-authored an academic paper with my team members, published in the El-Cezeri Journal of Science and Engineering (2026), on deep learning models integrating attention mechanisms (Attention U-Net/U-Net++) for military camouflaged object detection. During my internship at Deep Think Technology, I developed industrial computer vision systems featuring real-time YOLOv8-seg inference, PyQt5 GUI design, and camera SDK integration. I am experienced in optimizing Small Language Models (SLMs like mT5) and Edge AI/TinyML applications (SVM on Arduino) using ONNX Runtime and INT8 quantization for resource-constrained devices. I ranked in the top 10 finalists of the Pupilica AI Hackathon with the practical solutions I developed. My primary focus lies in end-to-end model development workflows—from data preprocessing and model training to deployment and optimization in computer vision and natural language processing.

Recent Works

Arduino PC System Monitor

A TinyML-powered system that monitors PC hardware resources (CPU, RAM, GPU, VRAM) and automatically changes the power mode by classifying the current load using an SVM model running on an Arduino Leonardo.

Apple Detection and Quality Analysis System

An industrial desktop application built with PyQt5 for real-time apple detection, physical dimension measurement, and CIEDE2000 color quality analysis using YOLOv8-seg. Developed during an internship, I focused on designing the GUI and integrating the core system modules.

mT5 Bidirectional EN-TR Machine Translation (ONNX INT8 & ORT Mobile)

Fine-tuned an mT5-small model with 8-bit LoRA for bidirectional English-Turkish translation (EN<->TR). The model was optimized using ONNX and INT8 quantization specifically for mobile deployment via ONNX Runtime (ORT Mobile). Achieved a METEOR score of 0.6245 while significantly reducing VRAM usage.

Zettelkasten AI Notes

I built this desktop application with PyQt5 to make knowledge management more efficient using the Zettelkasten method. It allows you to create, edit, and link notes with full Markdown support and real-time previews, storing everything locally in an SQLite database. I also integrated Google Gemini AI to automatically generate Zettelkasten-style notes from PDF documents and suggest relevant links between your notes.
